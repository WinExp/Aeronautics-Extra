# Aeronautics Extra
This mod is designed to add practical features to Create Aeronautics.

**This project is in its early stages; the current version only implements additional peripherals from CC:Tweaked.**

**If you have any ideas or feedback, please feel free to submit an issue or pull request.**

For more information, please visit the [Wiki](https://github.com/WinExp/Aeronautics-Extra/wiki).

## Features
### CC:Tweaked Compat
#### Create Aeronautics
- Analog Transmission: Stepless speed regulation support.
- Hot Air Burner: Adjustable hot air amount support.
- Steam Vent: Adjustable steam amount support.

#### Create Propulsion: Simulated
- Thruster: Adjustable power support.
