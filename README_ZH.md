# Aeronautics Extra
本模组致力于向机械动力：航空学添加实用功能。

**本项目处于早期阶段，目前版本仅实现了 CC:Tweaked 的更多 Peripherals。**

**如果你有想法/意见，欢迎提交 Issue / Pull request。**

更多信息请查看 [Wiki](https://github.com/WinExp/Aeronautics-Extra/wiki)。

## 特性
### CC: Tweaked 兼容
#### 机械动力：航空学
- 模拟传动器: 无级调速支持。
- 热空气燃烧室: 可调最大热空气提供量。
- 蒸汽喷口: 可调最大蒸汽提供量。
